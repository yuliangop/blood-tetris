# 血之方块 · Blood Tetris

暗黑血腥风格的俄罗斯方块网页游戏。

## 特色
- 暗黑哥特式视觉风格，血腥粒子特效
- 7种暗黑主题方块：骨、眼、印、血、影、牙、爪
- 行消除血溅动画、屏幕震动（Tetris）
- 历史最高分记录（LocalStorage 持久化）
- 音效反馈（Web Audio API）
- 支持移动/旋转/硬降/持有等完整操作

## 操作
| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 顺时针旋转 |
| Z | 逆时针旋转 |
| ↓ | 软降 |
| 空格 | 硬降（直接落底） |
| C | 持有方块 |
| P | 暂停/继续 |
