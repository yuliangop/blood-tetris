# 僵尸方块 · Zombie Tetris

僵尸身体部件俄罗斯方块桌面游戏。

## 特色
- 墓地毒雾视觉风格
- 7种僵尸身体部件方块：高僵、头、躯、臂、影臂、腿、影腿
- 行消除毒雾爆炸动画、Tetris 屏幕震动
- 植物大战僵尸原声音效
- 历史最高分记录（LocalStorage 持久化）
- 支持移动/旋转/硬降/持有等完整操作
- 可打包为 Windows .exe 单机游戏

## 操作
| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 顺时针旋转 |
| Z | 逆时针旋转 |
| ↓ | 软降 |
| 空格 | 硬降（直接落底） |
| C | 持有方块 |
| P | 暂停/继续 |

## 运行

**方式一：浏览器直接运行**
双击 `index.html`

**方式二：桌面应用**
```bash
npm start        # 启动 Electron 桌面版
npm run build    # 打包为 Windows .exe（输出到 dist/）
```
